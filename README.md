

# SurfaceLab – COMP3015 Coursework 2

SurfaceLab is a real-time OpenGL shader showcase developed for COMP3015 Coursework 2.  
It is built on the Lab 1 template and demonstrates a set of advanced rendering techniques.

---

## ✨ Features

- Physically Based Rendering (PBR)  
  Implements realistic material lighting using physically accurate shading models.

- Shadow Mapping  
  Real-time shadows generated from a light source to enhance scene depth.

- Procedural Noise Material  
  Uses noise functions to generate dynamic surface detail without textures.

- Particle Fountain  
  A dynamic particle system simulating a fountain effect.

- Texture Sampling  
  Applied textures to enhance realism of surfaces.

- Shell-based Hair Rendering  
  Procedural layered shells used to simulate fur/hair appearance.
