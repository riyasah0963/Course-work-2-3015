# SurfaceLab

SurfaceLab is the COMP3015 Coursework 2 submission in this workspace. It is a
real-time OpenGL shader showcase built on the required Lab 1 template and
centred on three clear CW2 advanced features: physically based rendering,
shadow mapping, and a procedural noise material. The scene also includes a
particle fountain, texture sampling, and a shell-based procedural hair effect
to strengthen the overall presentation.

## What To Open

- Solution: `Lab 1/Project_Template.sln`
- Main scene: `Lab 1/scenebasic_uniform.cpp`
- Main shaders: `Lab 1/shader/basic_uniform.vert`, `Lab 1/shader/basic_uniform.frag`

## Controls

- `Esc`: quit
- `Space`: toggle scene animation

## Build Notes

Build `Debug|x64` or `Release|x64` from the `Lab 1` working directory. The
project expects the public OpenGL bundle at:

- `C:\Users\Public\OpenGL\include`
- `C:\Users\Public\OpenGL\lib`
- `C:\Users\Public\OpenGL\src`
